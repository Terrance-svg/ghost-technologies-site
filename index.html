Privacy is the architecture

Your vault should not become someone else’s database.

Ghost Inventory is built around device-side protection and minimized readable cloud metadata. Paid cloud backups encrypt sensitive vault content before upload.

  • ✓Client-side encrypted backupSensitive paid-plan vault content is designed to be encrypted on the device before upload. Ghost Technologies does not store the plaintext recovery passphrase or a server-side key intended to decrypt the private-vault backup.
  • ✓Intentional AI processingAI Scan processes only the image a signed-in user deliberately submits for analysis.
  • ✓Operational metadata onlyAccount status, plan, usage and synchronization timestamps can be used to operate the service without exposing inventory content.
